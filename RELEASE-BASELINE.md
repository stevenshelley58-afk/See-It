# Release Baseline

Baseline date: 2026-06-19.

The old Remix/Prisma/PromptOps split was deleted. The active build is a single Next.js App Router application backed by Supabase schema contracts, durable jobs, AI router/provider adapters, prompt control records, render trace records, Shopify app proxy routes, and founder operations surfaces.

Release candidate requires pnpm verify, dev store install, theme app extension install, PDP Lighthouse delta <= 10, and manual fixture review.
